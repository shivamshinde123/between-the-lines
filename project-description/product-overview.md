# Product Overview

## Name

Between the Lines

## Product Summary

Between the Lines is a personal book tracking web app with a cozy mystic library vibe. A user logs books they have read, uploads cover images, and writes free-form thought entries while reading. The app also generates reflective AI insights from those entries.

## Primary Goal

Create a private reading journal that combines structured book tracking with evolving AI-generated literary self-reflection.

## Core User Experience

1. Sign up and log in.
2. Add books with title, author, and cover image.
3. Browse books alphabetically from the home page.
4. Search books by title in real time.
5. Open a book detail page and write timestamped thought entries over time.
6. Edit or delete books and entries with confirmation prompts.
7. Generate AI reflections for a single book and across the full library.

## Product Principles

- Private by default: every user sees only their own library and thoughts.
- Reflective, not social: no sharing features in the initial version.
- Functional first: visual polish can evolve after the main flows are stable.
- Persistent context: project state should be recorded both in-repo and in Obsidian.
